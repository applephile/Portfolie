앱과 웹의 차이점 
앱 : 디바이스에 '설치'해서 사용 
웹 : '설치없이' 브라우저를 통해 바로 사용

모바일
- 다양한 기기의 사이즈와 화면 해상도를 가진다
- 모바일 작업의 경우 가로 사이즈의 기본 단위는 %로 작성함
- 내가 입력하는 크기가 모든 환경에서 동일하게 적용되도록 meta태그 
중 뷰포트 정보를 반드시 입력해야한다

경로를 담고 있는 속성
<img src="이미지 경로">
<a href="이동할 경로"></a>

href : href에 지정된 속성값을 사용했다면 히스토리로 내 컴퓨터에 캐쉬로 남음
src : 접속할 때마다 서버로부터 계속해서 데이터를 다운로드 받는다
